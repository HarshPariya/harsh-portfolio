## ✨ About This Project

A high-performance, visually stunning **developer portfolio** built from scratch to showcase my projects, skills, and experience. Designed with a focus on **speed**, **accessibility**, and **premium aesthetics**.

### Key Highlights

| Feature | Description |
|---------|------------|
| ⚡ **Zero-Bundle Animations** | Smooth, complex animations without impacting the main thread |
| 📱 **Fully Responsive** | Pixel-perfect across mobile, tablet, and 4K desktop |
| 🏎️ **High Performance** | Server-side rendering with Next.js 15 App Router |
| 🎨 **Premium Design** | Custom design system with smooth transitions & micro-interactions |
| 🔍 **SEO Optimized** | Structured data, metadata, sitemap, and Open Graph tags |
| 🖱️ **Custom Cursors** | Interactive splash cursor and dot cursor effects |
| 📊 **Live GitHub Stats** | Real-time GitHub contribution calendar and repo showcase |
| 💻 **LeetCode Integration** | Live coding profile stats |

## 💻 Getting Started

```bash
# 1. Clone the repository
git clone https://github.com/HarshPariya/harsh-portfolio.git
cd harsh-portfolio

# 2. Install dependencies
npm install

# 3. Start the development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

### Other Scripts

| Command | Description |
|---------|------------|
| `npm run build` | Production build |
| `npm run lint` | Run ESLint |
| `npm run format` | Format code with Prettier |
| `npm run test` | Run Playwright E2E tests |
| `npm run type-check` | TypeScript type checking |

## 📂 Project Structure

```
harsh-portfolio/
├── app/
│   ├── (site)/             # Page routes (Home, Projects, Coding Profiles)
│   ├── components/         # Reusable UI components
│   ├── data/               # Project data, review data, LeetCode data
│   ├── hooks/              # Custom React hooks
│   ├── ui/                 # Core UI primitives
│   ├── utils/              # Utility functions & lazy loaders
│   ├── layout.tsx          # Root layout with metadata & analytics
│   └── globalsV2.css       # Global styles
├── config/
│   ├── siteConfig.ts       # Site metadata, NAP, social links
│   └── schemas.ts          # JSON-LD structured data
├── public/                 # Static assets (images, icons, resume)
├── e2e/                    # Playwright end-to-end tests
├── scripts/                # Build & analysis scripts
└── package.json
```
